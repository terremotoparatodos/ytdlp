yt-dlp

clic para copiar yt-dlp -f "bestvideo+bestaudio" --merge-output-format mp4 --embed-metadata -o "%(title)s.%(ext)s" "URL_AQUÍ"
Instalación
1
Instalá yt-dlp:
Mac → brew install yt-dlp
Windows → winget install yt-dlp o bajalo desde github.com/yt-dlp
2
Instalá ffmpeg (necesario para unir video y audio):
Mac → brew install ffmpeg
Windows → winget install ffmpeg
3
Pegá la URL arriba, copiá el comando y pegalo en la terminal. El archivo queda en la carpeta donde estés parado.
4
Para elegir en qué carpeta guardar, agregá la ruta en el -o:
-o "/Users/vos/Videos/%(title)s.%(ext)s"
¡Copiado!
